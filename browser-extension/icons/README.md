# Browser Extension Icons

The extension needs icons in these sizes:

- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

## How to Create Icons:

### Option 1: Use Online Tool

1. Go to https://www.favicon-generator.org/
2. Upload a logo or create one
3. Generate icons in 16x16, 48x48, 128x128
4. Download and place in `browser-extension/icons/` folder

### Option 2: Use Design Tool

1. Create a simple logo in Canva/Figma
2. Export in 16x16, 48x48, 128x128
3. Save as PNG files
4. Place in `browser-extension/icons/` folder

### Option 3: Use Emoji (Quick)

1. Use an emoji screenshot tool
2. Capture ⚡ emoji
3. Resize to 16x16, 48x48, 128x128
4. Save and use
