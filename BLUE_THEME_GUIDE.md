## 🎯 **Color Usage Guide**

### **When to Use Each Color:**

**Navy Blue (`blue-800`)**:

- Primary text in gradients
- Important headings
- Brand identity

**Blue (`blue-600`)**:

- Primary buttons
- Icons
- Active states
- Links

**Sky Blue (`sky-500/600`)**:

- Gradient endings
- Secondary accents
- Hover states

**Light Blue (`blue-100`)**:

- Active backgrounds
- Subtle highlights
- Borders

**White**:

- Card backgrounds
- Navigation
- Clean surfaces

**Gray Tones**:

- Body text (`gray-900`)
- Secondary text (`gray-600`)
- Borders (`gray-200`)

## 🎨 **Gradient Formulas**

### **Primary Brand Gradient**:

```css
from-blue-800 to-sky-600
/* Navy blue → Sky blue */
```

### **Button Gradient**:

```css
from-blue-600 to-sky-500
/* Blue → Sky blue */
```

### **Background Gradient**:

```css
from-slate-50 via-blue-50 to-sky-50
/* Soft gray → Light blue → Sky tint */
```
